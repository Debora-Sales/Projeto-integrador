# 🧹 Xô Sujeira - Sistema de Gestão Empresarial

Sistema desenvolvido pela **GET (Gestão Empresarial Tech)** como Projeto Integrador do Curso Técnico em Informática do SENAC Jaboticabal.

O objetivo do sistema é automatizar processos administrativos e operacionais de pequenas empresas do segmento de produtos de limpeza, oferecendo controle de estoque, vendas, clientes, funcionários, logística e relatórios gerenciais.

## 📌 Sobre o Projeto

A empresa fictícia **Xô Sujeira** é uma loja de produtos de limpeza que necessitava modernizar seus processos internos.

Antes da implantação do sistema, atividades como controle de estoque, cadastro de clientes, gestão de vendas e geração de relatórios eram realizadas manualmente, resultando em perda de tempo, inconsistências de dados e dificuldade no acompanhamento do negócio.

O sistema foi desenvolvido para solucionar esses problemas através de uma interface simples, intuitiva e compatível com computadores de hardware mais antigo.

## 🔐 Controle de Acesso

O sistema trabalha com três perfis distintos:

### Administrador

Possui acesso total ao sistema:

* Estoque
* Financeiro
* Relatórios
* Cadastros
* Auditorias

### Estoquista

Possui acesso a:

* Cadastro de produtos
* Controle de estoque
* Movimentação de materiais

Restrições:

* Não visualiza valores financeiros
* Não pode excluir produtos

### Vendedor

Possui acesso a:

* Cadastro de clientes
* Registro de vendas
* Consulta de estoque

Não possui acesso ao módulo de controle de estoque.
