# GET - Gestão Empresarial Tech

Site institucional desenvolvido para apresentação da empresa **GET (Gestão Empresarial Tech)** e do software **Xô Sujeira**, criado como parte do Projeto Integrador do Curso Técnico em Informática do SENAC Jaboticabal.

---

## 📋 Sobre o Projeto

A GET é uma empresa fictícia especializada em soluções tecnológicas para gestão empresarial.

Seu principal produto é o **Xô Sujeira**, um sistema voltado para empresas do segmento de produtos de limpeza, oferecendo recursos de:

* Controle de estoque
* Controle de lotes e validade
* Gestão de vendas
* Logística B2B
* Controle de comissões
* Cadastro de clientes e vendedores
* Relatórios gerenciais

O objetivo deste site é apresentar a empresa, divulgar o software e disponibilizar um canal de contato para potenciais clientes.

## 🧹 Sobre o Xô Sujeira

O sistema foi projetado com base no levantamento de requisitos realizado durante a fase de análise do Projeto Integrador.

### Requisitos Funcionais

* Cadastro de produtos com lote e validade
* Cadastro de clientes
* Cadastro de vendedores
* Controle de estoque mínimo
* Venda por unidade
* Relatórios de comissão
* Gestão de frete

### Regras de Negócio

* Bloqueio de venda de produtos vencidos
* Bloqueio de estoque negativo
* Controle de acesso por perfil
* Desconto para pagamentos à vista
* Relatórios financeiros restritos ao administrador
